# Rotorflight Radio Updater

This folder contains the Windows updater for the Rotorflight Lua Ethos Suite.

## Download

Windows executable:

`https://raw.githubusercontent.com/rotorflight/rotorflight-lua-ethos-suite/master/bin/updater/update_radio_gui.exe`

## Release Metadata

The updater checks for updates by loading:

`bin/updater/src/release.json`

Update that file when you publish a new updater version.
